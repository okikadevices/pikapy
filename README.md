# pikapy
Python configuration of Okika Pi.Ka board

spitest.py: Python code to test SPI configuration of the FPAAs on the Pi.Ka board (works for Raspberry Pi 4 and earlier)

spitestPi5.py: Python code to test SPI configuration of the FPAAs on the Pi.Ka board (works for Raspberry Pi 5)
4osc.ahf: Sample configuration file that includes 4 sine wave oscillators at 80kHz, 150kHz, 277kHz, and 493kHz

See PDF file for programming instructions
